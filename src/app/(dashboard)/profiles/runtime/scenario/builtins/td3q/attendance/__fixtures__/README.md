# TD3Q Attendance Fixtures

This directory contains screenshot fixtures used to verify the TD3Q attendance scenario without launching a real browser session.

## Expected Fixture Names

- `attendance-popup-open.png`: A screenshot showing the daily attendance popup fully open.
- `daily-ready.png`: A screenshot showing the daily reward available to claim today.
- `daily-done.png`: A screenshot showing today's attendance has been fully claimed.
- `daily-reward-popup-open.png`: A screenshot showing the small "Daily Reward" popup box.
- `milestone-ready.png`: A screenshot showing a milestone reward ready to be claimed on the bottom progress bar.
- `close-ready.png`: A screenshot showing the popup close button (X) is visible and interactable.

If these screenshots are provided, the fixture verification harness can run detectors locally. Missing files will be skipped automatically without error.
